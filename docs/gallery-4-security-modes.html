Security Enforcement Modes

🚫
Block
SECURITY_PII_MODE=block
Returns HTTP 403 immediately when PII is detected. Request never reaches the LLM.
Strictest
🔒
Redact
SECURITY_PII_MODE=redact
Auto-scrubs PII from prompt before forwarding. LLM receives clean text. Response scanned too.
Recommended
📝
Log
SECURITY_PII_MODE=log
Logs PII findings but does not block or modify. Good for audit-only deployments.
Permissive

Same modes for injection detection:

Block

SECURITY_INJECTION_MODE=block
Returns 403 on prompt injection attempts.

Log

SECURITY_INJECTION_MODE=log
Logs injection findings without blocking.

Webhook Alerts

SECURITY_WEBHOOK_URL=https://...
Fires JSON to Slack/PagerDuty on any security event.
